### 安装使用

#### 从 `Github`拉取代码

```bash
git clone https://github.com/snailuu/kuku-fronted.git
```



#### 安装依赖

```bash
cd kuku-fronted
npm i
```

#### 启动平台

```bash
npm run dev
```

#### 项目打包

```bash
npm run build
```
